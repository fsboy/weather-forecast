# Weather Forecast Skill 测试说明

## 问题修复总结

### 原始问题
CoPaw调用skill时出现错误：
```
Command failed with exit code 127.
/bin/sh: 1: weather-forecast: not found
```

### 根本原因
根据CoPaw官方文档，Skill的正确结构是：
1. **Skill只需要SKILL.md文件**：CoPaw会自动识别包含SKILL.md的目录
2. **SKILL.md是给Agent看的能力说明**：不是命令行工具
3. **工具函数应该在Python模块中定义**：供Agent调用

### 修复内容

#### 1. 添加命令行入口点（可选）
- 创建了 `__main__.py` - Python模块入口
- 创建了 `weather-forecast` - 可执行脚本（Linux/Mac）
- 修改了 `weather_tools.py` 的 `__main__` 部分，添加argparse支持

#### 2. 更新SKILL.md
添加了详细的工具调用指南：
- 说明了三个异步函数的调用方式
- 添加了Agent使用指南
- 明确了不同场景下应该使用哪个函数
- 提供了代码示例

## Skill结构

```
weather-forecast/
├── SKILL.md              # 技能描述（必需）
├── weather_tools.py      # Python工具实现
├── __main__.py          # Python模块入口（可选）
├── weather-forecast      # 可执行脚本（可选）

```

## CoPaw集成方式

### 方式一：作为Skill使用（推荐）

1. **复制到CoPaw目录**
   ```bash
   cp -r weather-forecast ~/.copaw/customized_skills/
   ```

2. **配置API密钥**
   ```bash
   # 方式A：环境变量
   export AMAP_API_KEY="your_api_key"
   
   # 方式B：config.json
   # 编辑 ~/.copaw/config.json
   {
     "env": {
       "AMAP_API_KEY": "your_api_key"
     }
   }
   ```

3. **重启CoPaw**
   ```bash
   copaw app --reload
   ```

4. **在对话中使用**
   ```
   用户：北京今天天气怎么样？
   用户：沈阳浑南区明天天气如何？
   ```

### 方式二：作为命令行工具使用

```bash
# 实时天气
python weather_tools.py --city "沈阳浑南区"

# 今日天气
python weather_tools.py --city "北京" --type today

# 天气预报
python weather_tools.py --city "上海" --type forecast
```

## Agent调用示例

当Agent需要查询天气时，应该：

```python
# 导入工具
from weather_tools import get_realtime_weather, get_forecast_weather, get_today_weather

# 查询实时天气
result = await get_realtime_weather("沈阳浑南区")
if result["success"]:
    print(f"{result['city']}当前天气：{result['weather']}")
    print(f"温度：{result['temperature']}°C")

# 查询今日天气
today = await get_today_weather("北京")
if today["success"]:
    print(f"今天：{today['day_weather']} {today['day_temp']}°C")

# 查询天气预报
forecast = await get_forecast_weather("上海")
if forecast["success"]:
    for day in forecast["forecasts"]:
        print(f"{day['date']}: {day['day_weather']}")
```

## 获取API密钥

1. 访问 [高德开放平台](https://console.amap.com/)
2. 注册并登录
3. 创建应用 → 选择"Web服务"
4. 获取Key

## 注意事项

1. **API密钥安全**：不要硬编码在代码中，使用环境变量或config.json
2. **请求频率**：高德API有调用频率限制
3. **城市编码**：支持城市名自动转换为adcode
4. **异步调用**：所有工具函数都是异步的，需要使用await
